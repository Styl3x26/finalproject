Proiect realizat cu ajutorul React and MySQL.
Pentru conexiunea cu baza de date se intra in backend > index.js si se editeaza linia 10 - 13 cu datele noastre de la baza de date.
Baza de date o aveti in folder.
In consola se tasteaza node index.js, se deschide o alta consola in folderul CLIENT, in care tastam npm start. ( trebuie sa avem instalat npm, daca nu l avem, tastam npm install )
Accesam localhost:3000.
